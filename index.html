    lightrag-slim¶

    A lightweight, standalone graph-based RAG engine — extracted from LightRAG.

    Pure Python · zero server dependencies · bring your own LLM.

    Get Started Quick Start GitHub

    Features¶

    • Graph-Based Retrieval

      Builds a knowledge graph from documents to answer complex multi-hop questions that simple vector search misses.

    • 8 Dependencies

      Reduced from 18+ in full LightRAG. No database server, no vendor SDK required.

    • Bring Your Own LLM

      Plug in any LLM or embedding model — OpenAI, DeepSeek, Anthropic, local Ollama, or anything else.

    • File-Based Storage

      All data stored as plain files — NetworkX graph, NanoVectorDB, JSON. No infrastructure needed.

    • 4 Query Modes

      naive / local / global / hybrid — from simple vector search to full graph traversal.

    • Pure Async Python

      Built entirely on asyncio. Drop into any Python application or agent framework.

    Acknowledgements¶

    Built on the shoulders of giants

    lightrag-slim is a derivative of LightRAG — a brilliant graph-based RAG framework developed by researchers at The Hong Kong University of Science and Technology. The core retrieval algorithm, entity/relation extraction pipeline, multi-mode query system, and cross-process shared storage design all originate from their exceptional work.

    If this library is useful to you, please ⭐ star the original LightRAG repository and consider citing their paper.
